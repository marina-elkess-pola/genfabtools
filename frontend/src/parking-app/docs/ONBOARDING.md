# GenFabTools Parking Engine — First-Time User Onboarding

---

## Welcome Message

**Welcome to the GenFabTools Parking Engine.**

This tool helps you estimate how many parking stalls fit on a site — quickly and transparently. Enter your site dimensions, choose a configuration, and get a conceptual stall count in seconds. All assumptions are shown so you can understand exactly how the result was calculated.

---

## How to Use This Tool

**Getting started takes less than a minute:**

1. **Enter your site dimensions**  
   In the left panel, type the width and height of your site in feet, then click "Create Site."

2. **Choose your aisle type**  
   Select One-Way or Two-Way aisles. Two-way is more common but uses more space.

3. **Set your setback**  
   Enter the distance from the property line where parking cannot be placed.

4. **Review the results**  
   The right panel shows your total stall count, including ADA stalls, plus efficiency metrics.

5. **Check how the result was generated**  
   Scroll down to see the exact assumptions (stall size, aisle width) that produced your number.

---

## Important Disclaimer

**This tool provides conceptual estimates for early-stage feasibility only.**

- Results are based on generic U.S. parking standards, not local codes.
- Stall counts are approximate and may differ from final engineered layouts.
- This tool does not certify ADA compliance or code conformance.
- Output is not suitable for permit applications or construction documents.

Use this tool to inform early decisions — not to replace professional design review.

---

## Quick Reference

| Input | What It Does |
|-------|--------------|
| Site Dimensions | Defines the area available for parking |
| Aisle Direction | One-way (narrower) or Two-way (wider, more flexible) |
| Setback | Distance from edge where parking is not allowed |

| Output | What It Means |
|--------|---------------|
| Total Stalls | Estimated count of parking spaces that fit |
| ADA Stalls | Accessible stalls calculated per generic standards |
| Efficiency (SF/stall) | Square feet of site used per stall |
| Usability Ratio | Percentage of site that can hold parking |

## Need Help?

If results seem unexpected:

- Check the "Why not more stalls?" hint for the main limiting factor
- Review the "How this result was generated" panel for assumptions
- Try adjusting setback or aisle direction to compare scenarios
