Residential Scheme Intelligence

Analyze residential scheme efficiency, visualize performance diagnostics, compare design alternatives, and export professional reports — all inside Autodesk Revit.

1. Installation

RSI is distributed as a standalone Windows installer. No manual file copying is required.

System Requirements

  • Autodesk Revit 2024
  • Windows 10 or later (64-bit)
  • .NET Framework 4.8

Install Steps

  1. Close Revit if it is currently running.
  2. Run RSI_Setup.exe. The installer requires administrator privileges to write to the shared Revit Addins folder.
  3. Follow the on-screen prompts. The installer places all plugin files into:
    C:\ProgramData\Autodesk\Revit\Addins\2024\
  4. Launch Revit. A new GenFabTools tab will appear in the ribbon with the RSI button.
On first launch, RSI will prompt you to enter a license key and display a welcome dialog.

Uninstalling

Open Settings → Apps & Features, find Residential Scheme Intelligence, and click Uninstall. All plugin files will be removed.

2. Quick Start

Follow these steps to run your first analysis:

  1. Open an Area Plan
    RSI works exclusively with Revit Area Plans. Open an existing Area Plan view or create one via Architecture → Area Plan. The active view must be an Area Plan for analysis to proceed.
  2. Assign RSI Categories
    Each area in your plan needs an RSI_Category classification. Categories include:
    Category Description
    Net Sellable Residential units, private amenity
    Core Stairs, lifts, risers, lobbies
    Circulation Corridors, hallways, shared access routes
    Parking Vehicle parking areas
    Amenity Shared resident amenities (gym, lounge)
    BOH Back-of-house, plant rooms, storage
    You can assign categories manually or use RSI's Auto-Classify feature.
  3. Click Analyze Project
    Click the RSI button in the GenFabTools ribbon tab. The dockable panel opens on the right with a full breakdown of your scheme metrics: Gross Area, Net Sellable Area, Core, Circulation, and the resulting Efficiency Ratio.
Use the Scope dropdown to analyze the active Area Plan or all Area Plans in the project.

3. Heatmap Analysis

The Performance Diagnosis heatmap provides an immediate visual layer over your Area Plan, highlighting areas that negatively impact scheme efficiency.

How It Works

When you activate the heatmap via the Show Heatmap button in the Analyze tab, RSI creates color-coded FilledRegion overlays directly on your view:

Color Meaning
Red Core Over-Allocation — Core areas exceed the benchmark threshold for the selected building type.
Amber Circulation Over-Allocation — Circulation space is disproportionately large relative to Net Sellable area.

Clearing the Heatmap

Click Clear Heatmap to remove all RSI-generated FilledRegions (identified by the RSI_Heatmap_ prefix). The underlying areas are unaffected.

The heatmap overlays are view-specific. Switching to a different Area Plan will not carry heatmap data across views. Re-analyze and re-apply as needed.

4. Scheme Comparison

RSI lets you save analysis snapshots and compare design alternatives side by side.

Saving a Scheme

  1. Run an analysis on your current design.
  2. Switch to the Compare tab.
  3. Enter a name (e.g., Option A — Dual Core) and click Save Scheme.

The snapshot captures Gross Area, Net Sellable, Core, Circulation, Efficiency, and per-category area counts at that moment. Saved schemes are stored in the Revit project via Extensible Storage, so they persist with the model file.

Comparing Schemes

  1. Select Scheme A and Scheme B from the dropdowns.
  2. Click Compare.

RSI displays a delta table showing the difference in every metric between the two saved schemes, making it straightforward to evaluate which layout performs better.

Comparison reads stored values only — it never recalculates from the live model. This ensures you are comparing the exact state at the time each scheme was saved.

5. Export Reports

RSI supports three export formats, each generated from the cached analysis data.

Format Contents Use Case
Excel (.xlsx) Four worksheets: Current Analysis, Category Breakdown, Saved Schemes, Comparison Detailed data analysis, client reporting, further spreadsheet work
PDF Formatted multi-page report with summary metrics and category tables Stakeholder presentations, design reviews, printable reports
CSV Flat comma-separated values of key metrics Data interchange, import into other tools

Exporting

  1. Ensure an analysis has been run (the Analyze tab shows results).
  2. Click the export button for your desired format.
  3. Choose a save location in the file dialog. Default filenames include a timestamp (e.g., RSI_Report_20260317.xlsx).
Exports read from cached data and do not trigger a new analysis. If you have modified the model since the last analysis, click Re-Analyze first to refresh the data.

6. Troubleshooting

Heatmap Not Showing

  • Ensure you are viewing an Area Plan. Heatmap overlays are created as FilledRegions on the active Area Plan view and will not appear in floor plans or 3D views.
  • Check that an analysis has been run first. The heatmap button is only active after results are available.
  • Verify that the active view's Visibility/Graphics has Filled Regions enabled (category must be visible).
  • If a previous heatmap is stuck, click Clear Heatmap and then Show Heatmap again.

Areas Not Categorized

  • RSI reads the RSI_Category parameter on each Area element. If this parameter is missing or empty, the area will appear as Unassigned.
  • Use Auto-Classify to let RSI assign categories based on area naming conventions.
  • For manual assignment, select one or more areas in Revit and use the category assignment controls in the RSI panel.
  • Click Highlight Unclassified to visually locate areas that still need a category.

Export Not Working

  • Ensure an analysis has been completed before exporting. If the panel shows no results, click RSI in the ribbon to analyze first.
  • If saving an Excel file fails, make sure the target .xlsx file is not already open in another application.
  • Check that you have write permissions to the chosen save location.
  • For PDF exports, ensure PdfSharp.dll is present alongside the main plugin DLL. Reinstalling via RSI_Setup.exe will restore any missing files.
